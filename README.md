GymTracker is a desktop application built using Python that allows users to log workouts, track performance, and analyze fitness progress using data visualization.

The application focuses on applying Data Science concepts to a real-world use case by transforming raw workout data into meaningful insights through graphs, analytics, and reports.

It provides a simple yet powerful interface for managing workout history while enabling users to monitor strength progression and training patterns over time.




- 📝 **Workout Logging**
  - Record exercises with sets, reps, and weight

- 📊 **Progress Tracking**
  - Visualize workout performance using line graphs

- 🥧 **Muscle Group Analysis**
  - Pie chart to show distribution of trained muscle groups

- 📈 **Training Volume Analysis**
  - Bar graph to compare workout intensity and volume

- 🧮 **1 Rep Max (1RM) Estimation**
  - Estimate maximum strength using workout data

- 🏆 **Personal Records Tracking**
  - Automatically track best performance

- 🗂️ **Workout History Management**
  - View and delete past entries

- 📄 **PDF Export**
  - Generate and export workout reports

- 💾 **Lightweight Data Storage**
  - Uses CSV for fast and simple data handling
 
  ## 🛠️ Tech Stack

- Python
- Tkinter (GUI)
- Pandas (Data Processing)
- Matplotlib (Data Visualization)
